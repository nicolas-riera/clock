# Mamie Jeannine’s Clock

## Description

This is a Python clock made for Grandma Jeannine, after her naughty cat broke her favorite clock.

## Requirements

- Python 3.10+

## Installation

Clone the repository:

```
git clone https://github.com/nicolas-riera/clock
```

## Usage

### How to run the script

Run the main script to start the clock:

```
python main.py
```

## Features

- Real-time clock display
- Alarm management
- Simple terminal menu interface
- Commands to start, stop, pause, etc
